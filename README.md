# MediConnect 🏥

### Doctor Appointment System (MERN Stack)

## Overview

MediConnect is a full-stack web application that helps patients book doctor appointments easily. It provides a simple and user-friendly platform for managing appointments and profiles.

---

## Features

* Patient registration and login
* Doctor registration and profile management
* Search doctors by category
* Book and manage appointments
* View appointment history
* Secure authentication system
* Responsive and easy-to-use interface

---

## Tech Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB

---

## How to Run

### 1. Install Dependencies

```bash
npm install
```

### 2. Setup Environment Variables

Create a `.env` file and add:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```

### 3. Run the Project

```bash
npm start
```

---

## Usage

* Open the app in browser (http://localhost:3000)
* Register as a user or doctor
* Book and manage appointments
